Fila do Pastel Sistema de abertura de chamados

Olá! Seja bem vindo ao sistema Fila do Pastel desenvolvido por mim no curso Desenvolvedor Full Stack.
Esse sistema é um sistema para gerenciamento de chamados simulando um simples Hel Desk desenvolvido em Angular e Spring Boot 3
O projeto tem como objetivo principal abordar os conceitos de aplicações SPAs, Autenticação e Automação, Tokens JWT e muito mais. O código fonto do projeto pode ser acessado clicando no link Github no meu repositorio

Fique a vontade para testar o sistema e deixar sua sugestão.
